# Traffic Control System

This project is a simulation of a traffic control system based on the ECEN723 Spring 2026 project overview.

## Building the Project

To build the project, you will need `cmake` and a C++ compiler.

```bash
mkdir build
cd build
cmake ..
make
```

## Running the simulations

```bash
# Run the infrastructure simulation
./build/infrastructure_sim

# Run the vehicle simulation
./build/vehicle_sim
```
